# Art Stimulus Set

Annotated Chilean art stimulus set from MAVI-UC (Museum of Visual Arts, Universidad Católica) used in the PhD project: Neural, Cognitive, and Experiential Dynamics of Aesthetic Experience through Visual Artworks.

## Access

The stimulus image files are hosted on Zenodo with restricted access:

**DOI:** [10.5281/zenodo.19560124](https://doi.org/10.5281/zenodo.19560124)

Request access there.

## Contents

- `metadata/` : Annotation files and codebook describing the artwork variables
- `permissions/` : Copyright permission documents from MAVI-UC
