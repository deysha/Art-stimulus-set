# Codebook
 i Columns in ArtworksMAVIUC.ods
    - #: number of rwos
    - ID_imagen: unique image identifier
    - ID_obra: artwork catalog identifier
    - Autor: artist name
    - Título: artwork title
    - Año: year of creation
    - Tipo_obra: artwork type/category
    - Técnica: technique/medium used
    - Dimensiones_cm: physical dimensions in centimeters
    - Notas: The name of the collection to which the work belongs (3 collections)
 ii Columns in ArtworkSet_Curated.xlsx
    - #: sequential stimulus numeber
    - artwork ID: unique image identifier.
